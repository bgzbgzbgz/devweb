Cara deploy Laravel di shared hosting?

oleh Adit • 2 hari lalu

Saya punya project Laravel dan mau coba deploy ke hosting biasa (bukan VPS/Cloud). Sudah coba upload via cPanel tapi halaman utamanya malah menampilkan struktur folder, bukan websitenya.

Sudah coba ubah file `.htaccess` juga tapi masih belum berhasil. Apakah ada langkah-langkah khusus yang perlu diperhatikan? Mungkin soal path `public` atau konfigurasi lainnya? Mohon pencerahannya, terima kasih!

15 Komentar

Biasanya, kamu perlu memindahkan semua isi dari folder `/public` Laravel ke `public_html` di cPanel, dan sisa file Laravel lainnya diletakkan di luar `public_html`.

oleh Rani • 2 hari lalu

Betul kata kak Rani. Setelah itu jangan lupa sesuaikan path di file `index.php` yang ada di `public_html` agar mengarah ke direktori Laravel yang benar.
